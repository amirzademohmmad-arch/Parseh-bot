# Parseh Cloud Telegram Bot

این نسخه با کتابخانه `requests` نوشته شده و برای نصب روی سرویس ابری ساده‌تر است.

## متغیرهای لازم

در پنل سرویس ابری این دو Environment Variable را وارد کنید:

BOT_TOKEN = توکن ربات تلگرام  
ADMIN_ID = آیدی عددی تلگرام شما

برای گرفتن ADMIN_ID به ربات @userinfobot پیام بدهید.

## دستور اجرا

python bot.py

## فایل‌های اصلی

- bot.py
- requirements.txt
- runtime.txt

## نکته امنیتی

توکن ربات را داخل چت عمومی یا فایل قابل مشاهده قرار ندهید.
