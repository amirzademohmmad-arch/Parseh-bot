# Parseh Telegram Bot

فایل‌های لازم:
- bot.py
- requirements.txt
- runtime.txt
- README_FA.md

## Render
Build Command:
pip install -r requirements.txt

Start Command:
python bot.py

## Environment Variables
BOT_TOKEN = توکن ربات از BotFather
ADMIN_ID = آیدی عددی تلگرام مدیر

- ثبت سفارش عمده
- پیگیری سفارش
- تغییر وضعیت سفارش
- حذف محصول
